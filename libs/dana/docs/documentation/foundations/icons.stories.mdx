# Icons

## Usage

Import an icon directly from **@dana/icons**. For example:

```typescript
import { SvgLinkedin } from '@dana/icons';
```

An icon allow the following parameters:

-   size: The size of the icon: Possible values are: **'xsmall' | 'small' | 'medium' | 'large' | any number**

<Preview>
    <SvgGithub size="large" />
</Preview>

<Preview>
    <SvgGithub size="48" />
</Preview>
