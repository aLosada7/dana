# Icons

## Usage

Import an icon directly from **@dana/icons**. For example:

```typescript
import { SvgLinkedin } from '@dana/icons';
```

An icon allow the following parameters:

-   size: The size of the icon: Possible values are: **'xsmall' | 'small' | 'medium' | 'large'**

<Preview>
    <GithubIcon size="small" />
</Preview>

<Preview>
    <GithubIcon size="large" />
</Preview>
